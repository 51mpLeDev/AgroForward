# Агрофорвард — сайт поставщика мясосырья

Тестовое задание: разработка сайта для B2B поставщика мясосырья с акцентом на надежность, масштаб и доверие.

---

## 📌 О проекте

Сайт разработан как презентационная платформа для компании, занимающейся поставками мясосырья для:

- агрохолдингов
- перерабатывающих предприятий
- розничных сетей

Основная задача — передать ощущение стабильности, прозрачности и профессионализма.

---

## 🎯 Подход к реализации

В процессе разработки я сделал упор на:

- **B2B-дизайн** — строгий, минималистичный, без “стартап-стиля”
- **Работу от логотипа** — цветовая система построена на:
    - `#d56154` — основной (действие, акцент)
    - `#04523a` — вторичный (доверие, стабильность)
- **UX-структуру** — логичная последовательность блоков
- **Анимации** — легкие и ненавязчивые, без перегрузки
- **Адаптивность** — корректная работа на всех устройствах

---

## 🧱 Структура сайта

- Hero — оффер и позиционирование
- About — информация о компании и миссия
- Stats — ключевые показатели
- Values — корпоративные ценности
- Timeline — этапы развития
- Investors — инвестиционная привлекательность
- Contact — форма обратной связи
- Footer — навигация и юридическая информация

---

## ⚙️ Технологии

- Nuxt 4
- Vue 3 (Composition API)
- Tailwind CSS
- Intersection Observer (анимации)

---

## ✨ Особенности реализации

- Собственный composable для анимаций (`useReveal`)
- Плавные появления элементов при скролле
- Анимированные счетчики (ключевые показатели)
- Чистая компонентная архитектура
- Использование blur + backdrop для глубины интерфейса

---

## 🎨 Дизайн

Дизайн выполнен в корпоративном стиле с акцентом на:

- контраст и читаемость
- минимализм
- ощущение “тяжелого” B2B сегмента

Основная идея — не перегружать интерфейс, а подчеркнуть надежность.

---

## 🚀 Запуск проекта

Требуется установленный Node.js версии 20 или выше.

```bash
node -v
```

Установка зависимостей и запуск:

```bash
npm install
npm run dev
```
